# Playwright-Demo

A simple demo repository with basic Node.js scripts and opencode tooling.

## Contents

- `playwrite-demo1/01-Basics.js` – Basic JavaScript console output examples.
- `playwrite-demo1/02_commands.js` – Node.js `process` information examples.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run a script:
   ```bash
   node playwrite-demo1/01-Basics.js
   node playwrite-demo1/02_commands.js
   ```

## Requirements

- [Node.js](https://nodejs.org/)
